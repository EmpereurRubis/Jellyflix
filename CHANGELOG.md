# Changelog

Toutes les modifications importantes apportées à ce projet seront documentées dans ce fichier.

Le format est inspiré de [Keep a Changelog](https://keepachangelog.com/fr/1.1.0/) et ce projet suit le versionnage sémantique (SemVer).

## [1.0.0] - 2026-07-07

### 🎉 Première version publique

### Ajouté

#### Interface générale
- Thème sombre inspiré de Netflix.
- Palette de couleurs personnalisée.
- Couleur principale rouge Netflix (`#E50914`).
- Variables CSS centralisées.
- Scrollbar personnalisée.

#### Navigation
- Barre de navigation modernisée.
- Couleur rouge sur les onglets actifs.
- Effets de survol personnalisés.
- Suppression des accents bleus Jellyfin.

#### Header
- Header semi-transparent.
- Effet de flou (Blur).
- Avatar utilisateur restauré et compatible avec Jellyfin 10.10+.

#### Boutons
- Boutons principaux personnalisés.
- Boutons de lecture rouges.
- Hover rouge sur les boutons.
- Hover rouge sur les icônes.

#### Cartes
- Animation d'agrandissement au survol.
- Suppression des bordures inutiles.
- Suppression des ombres.

#### Lecteur multimédia
- Barre de progression rouge.
- Barre restante grise.
- Curseur rouge.
- Slider de volume personnalisé.
- Compatibilité avec les nouveaux sliders Jellyfin 10.10.

#### Interface
- Badges rouges.
- Cases à cocher personnalisées.
- Suppression des contours bleus.
- Focus personnalisé.

#### Chargement
- Personnalisation du spinner Jellyfin.
- Optimisation des animations Material Design.
